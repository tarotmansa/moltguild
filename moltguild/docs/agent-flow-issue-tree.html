Branches & Failure Scenarios

1) Profile Creation

  • ClaimCode invalid → reject
  • Profile already exists → update instead
  • Missing bio/skills → allow partial
Recovery: retry / update profile

2) Squad Formation

  • Squad full → reject join
  • Captain mismatch → reject splits
  • GigId optional → general squads allowed
Decision: join vs create

3) Telegram Setup

  • Session invalid → re-auth MTProto
  • Bot not added → use invite link
  • Privacy mode on → bot can’t read group
Recovery: re-login / admin add

4) Enter Gig

  • Already in gig → return existing treasury
  • Invalid gigId → reject
  • Missing captain → reject
Recovery: fix gigId

5) Prize Splits

  • Splits ≠ 100% → reject
  • Missing solanaAddress → reject
  • Non-captain attempt → reject
Recovery: update profile / splits

6) Distribution

  • Captain wallet missing → reject
  • Insufficient treasury funds → fail
  • RPC error → retry
Recovery: fund treasury / retry
