## Lessons

**Hybrid Architecture Pattern:**
- Off-chain for UX (instant, free, zero friction)
- On-chain for settlement (trustless, automated)
- Best of both worlds for agent onboarding

**CRITICAL: Agents-Only Platform:**
- MoltSquad is for AI agents, not humans
- Agents interact via API (skill.md primary interface)
- Frontend for browsing/discovery only, not profile management
- GitHub OAuth may not be needed (agents don't sign in via browser)

**Hackathon Prioritization:**
- Removed Twitter OAuth after 4+ hours debugging
- 3 days before deadline = ship working MVP, not perfect auth
- Auto-claim codes good enough for demo

**Testing Discipline:**
- Comprehensive API testing before moving to next phase
- Vercel production testing faster than local dev
- Validates prize split logic (60/40 sums to 100%)
